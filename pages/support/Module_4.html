Module 4 is where you go on offense — carefully. You'll learn how attackers target credentials, crack password hashes using industry-standard tools, and analyze what those results mean for an organization's security posture. Week 1 focuses on Windows (NTLM), Week 2 on Linux (SHA-512), and Week 3 you deliver a professional password security assessment to two clients.

⚠️ Offensive Techniques Module: These skills are for authorized security assessments only. Never use password cracking tools without explicit written permission. Everything here is on NEXUS-authorized client systems.
