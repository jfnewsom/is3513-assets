Module 4 is where you go on offense — carefully. You'll learn how attackers target credentials, crack password hashes using industry-standard tools, and analyze what those results mean for an organization's security posture. Lab 4.1 focuses on Windows (NTLM) hashes from Brazos. Lab 4.2 moves to Linux (SHA-512) hashes from Alamo. Lab 4.3 you deliver a professional password security assessment to both clients.

⚠️ Offensive Techniques Module: These skills are for authorized security assessments only. Never use password cracking tools without explicit written permission. Everything here is on NEXUS-authorized client systems.
