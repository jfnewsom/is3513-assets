✓ 4. Set Up Your Kali VM

Your Kali Linux VM is your security workstation for the entire semester. Every lab runs on it. Set it up now and name it correctly.

VM naming convention: kali-lastname-studentID (e.g., kali-newsom-abc123). This name must appear in every screenshot.
