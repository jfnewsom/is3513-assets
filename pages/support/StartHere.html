Complete these 4 checkpoints before starting any labs

Complete these 4 checkpoints in order before diving into assignments. Each step ensures you’re set up for success.

✓ 1. Read the Syllabus

The syllabus is your contract for grading, policies, and expectations. It covers late work policy, drop policies, academic integrity, and everything else that governs this course. Find it in your Canvas course navigation (usually labeled “Syllabus” in the left sidebar).

✓ 2. Obtain the Textbook

Required: Conklin & White, Principles of Computer Security: CompTIA Security+ and Beyond, Sixth Edition (ISBN: 9781260474312). McGraw-Hill Education.

  • UTSA Bookstore — Physical or digital copies available
  • Amazon / other retailers — ISBN lookup for used copies; ensure 6th edition

✓ 4. Start Lab 1.1

Your Kali Linux VM is your security workstation for the entire semester. Setup happens inside Lab 1.1 — don’t set it up separately. Lab 1.1 walks you through installation, the VM naming convention, and the screenshot rules you’ll need for every assignment going forward. Find Lab 1.1 in your Canvas course modules.
