Step-by-Step Instructions

Step 1: Prepare Your Capture Environment

  1. Open Wireshark on your host machine (not inside the Kali VM)
  2. Identify your active network interface (Wi-Fi or Ethernet)
  3. Start a capture with filter: port 22

Step 2: Connect to Azure VM

  1. Open terminal in your Kali VM
  2. SSH to the Gulf Coast portal:
    ssh nexus-analyst@gulfcoast-portal.nexus-training.cloud
  3. Initial password: NEXUS-abc123! (case-sensitive)
  4. Accept the host key fingerprint when prompted

Step 3: Generate Analysis Traffic

  1. Once connected, explore the simulated patient portal environment
  2. Run the provided analysis scripts to generate representative traffic
  3. Allow capture to run for 5-10 minutes
  4. Stop the Wireshark capture and save as gulfcoast-live-capture.pcap
Common Pitfalls
  • VPN blocking SSH: If you can't connect, disable any VPN on your host machine first
  • Wrong interface: Capture on your HOST's interface, not inside Kali VM
  • Firewall issues: Some campus networks block outbound SSH (port 22) — try from a mobile hotspot
  • Password typo: Initial password is case-sensitive: NEXUS-abc123! (not nexus-ABC123!)

Network traffic analysis serves both offensive and defensive security operations. Understanding both perspectives makes you a more complete analyst.

What Red Team Learns

What can an attacker learn from this traffic? Your analysis revealed protocol distribution, active endpoints, and potentially exposed services. An attacker conducting reconnaissance would use these same techniques to map the attack surface.

What Blue Team Learns

What should defenders monitor? Your protocol hierarchy analysis establishes baseline "normal" traffic. Anomaly detection requires knowing what's expected — then anything outside that pattern becomes an alert candidate.

The Purple Team Connection

How do these perspectives inform each other? The Gulf Coast scenario demonstrates purple team value: analysis that serves both offensive reconnaissance (what can an attacker learn?) and defensive assessment (what should we monitor?). This dual perspective is exactly what NEXUS delivers.
