Step-by-Step Instructions

  1. Brazos Financial Recommendations (address NTLM weaknesses):
    • Consider Active Directory migration path away from NTLM
    • Implement password complexity AND length requirements
    • Deploy MFA for all privileged accounts immediately
    • Establish password rotation policy with breach monitoring
  2. Alamo Industries Recommendations (address shared accounts):
    • Eliminate shared supervisor account (CMMC requirement)
    • Implement individual accountability for all access
    • Strengthen already-good SHA-512 implementation
    • Document controls for CMMC evidence package
  3. Prioritize each recommendation using High/Medium/Low based on:
    • Risk severity (what happens if we don't fix it?)
    • Implementation complexity (how hard is this?)
    • Business impact (does this affect operations?)
